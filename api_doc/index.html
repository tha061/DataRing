DataRing: Privacy-Preserving Data Sharing Platform

A privacy-preserving data sharing platform enabling a set of organisations owning different datasets to perform data analysis on each other's dataset as well as joint analysis on multiple datasets in a controlled setting while ensuring the robustness of data inputs and query response and the confidentiality and privacy of data to be shared.
Below is an example of a DataRing system with three third-party servers and four participants each own a dataset. Each participant can evaluate its queries on other participants and respond to other participant's queries.
