Detailed Description

This class provides functions for a participant in the DataRing system.

The class implements functions for participant to convert its dataset in to a histogram, generate a partial view and compute query answer.

Generates a fake histgram from the true histogram.

To simulate a cheating participant in Partial View Collection, cheating participant generates a fake histgram from the true histogram by replacing n-keepDomainS by 0 and making n-keepDomainS dummy 1

Parameters
keepDomainSthe true records to be kept in the fake histogram
factorSizescale_up factor to limit the domain size to a*N
Returns
a fake histgram of size = dataset size * scale_up including only keepDomainS bins are true a map with aN elements of <label, binValue={1,0}>
