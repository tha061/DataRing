Detailed Description

Open API for EC-Elgamal crypto system.

Tha basic API is developed by Institute for Pervasive Computing, ETH Zurich and re-distributed by the authors of Dataring. DataRing reused basic encryption and developed more functions to use in DataRing.

Author
Lukas Burkhalter lubu@.nosp@m.inf..nosp@m.ethz..nosp@m.ch, Tham Nguyen tham..nosp@m.nguy.nosp@m.en@mq.nosp@m..edu.nosp@m..au, Nam Bui, DataRing
Date
April 2020
