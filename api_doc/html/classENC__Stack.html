Detailed Description

This class defines functions for pre-computating encryptions to improve performance.

Servers generate two stacks of pre-computed encryption of 1 and 0 and store them offline. Functions in this class including defining stack, initiate stack of encryption, pop an encrytion from the stack and re-fill extra encryption once the stack becomes empty.
