Detailed Description

This class provides functionalities of a single server in the DataRing system.

Functions include form an encrypted question from a query or a test, compute a answer for a test function from the encrypted partial view.

◆ prepareTestFuntion_Query_Vector()

void Server::prepareTestFuntion_Query_Vector ( ENC_Stack &  pre_enc_stack,
ENC_DOMAIN_MAP  enc_domain_map 
)

Pre-compute a test function or a query function.

This function pre-prepare a vector for a test function/query function, each element in the vector is a domain and its value is an enc(0). This is pre-computed so that when the server wishes to generate a test, it only needs to find the element where the domain is satisfied the condition of the test and changes it from enc(0) to enc(1)

Parameters
pre_enc_stackpoint to the precomputed encryption stack
enc_domain_mappartial view for server to get labels
Returns
a precomputed test function/query function with all enc(0)
