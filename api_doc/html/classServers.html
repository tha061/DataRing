Detailed Description

This class provides functions for servers to jointly perform activities in the DataRing system.

Functions include generating a collective public key from all servers; threshold decrypt of a ciphertext; creating a sampling vector for a participant to sample a partial view from its dataset; verfifying the partial view from a participant; verifying an answer for a test function from the participant.
